# ANX Usage Documentation

This directory contains usage documentation for ANX applications. When using ANX applications, you can find instructions and guides here.

## Available Documents

- **card-node-usage.md**: Guide for using ANX Card Node format (`<x kind cardKey>`) and CLI commands

## How to Use

1. **Card Node Format**: Use `<x kind cardKey>` tags to define ANX components
2. **CLI Commands**: Use `anx <cardKey> <action> <params>` to interact with components
3. **Refer to Documentation**: Check the specific documentation files for detailed usage instructions

For more information about ANX protocol, please refer to the main protocol documentation in the parent directories.
