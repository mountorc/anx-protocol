# ANX Skill Usage Guide

This file serves as a central reference to the usage documentation for ANX skill.

## Usage Documentation

All usage documentation is located in the `usage` directory. Here you can find detailed guides for different scenarios:

### Available Documentation

- **[How to Use ANX](usage/how-to-use-anx.md)**: Guide for using ANX Card Node format (`<x kind cardKey>`) and CLI commands to operate ANX applications

### How to Use

1. **Navigate to the usage directory**: `skill/usage/`
2. **Select the relevant guide** based on your use case
3. **Follow the instructions** provided in the specific documentation

For more information about the ANX protocol, please refer to the main protocol documentation in the parent directories.
